🎮 WHAT YOU'RE BUILDING

A Pokemon Go-style app where players collect tokens from real-world 
sightseeing locations using GPS. Features include:

✨ Features:
   🗺️  GPS-based token collection
   🎯  Collect tokens only when < 100m from sightseeing
   🏆  Virtual token collection
   📚  Collect tokens into themed sets
   🎁  Earn points and bonuses
   📊  Track your progress & statistics
   🏛️  Worldwide famous landmarks included
   ⚡ Real-time location updates

📞 TROUBLESHOOTING

Issue: "flutter: command not found"
→ Install Flutter SDK and add to PATH

Issue: "Location not updating"
→ Enable GPS in emulator
→ Grant location permissions
→ Check location service is on

Issue: "Maps API key error"
→ Get valid key from console.cloud.google.com
→ Enable Maps SDK for Android
→ Check internet connection

Issue: "Build fails"
→ Run: flutter clean
→ Run: flutter pub get
→ Try: flutter run again
